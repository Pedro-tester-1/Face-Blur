# Face Blur

Real-time face blurring using your webcam with OpenCV and MediaPipe.

## Technologies

- Python
- OpenCV
- MediaPipe

## Setup

Run the setup script once after cloning. It works on both Windows and Linux/macOS.

```bash
python setup.py
```

This will:
- Create a virtual environment named `face_blur`
- Install all dependencies from `requirements.txt`

Then activate the environment:

**Linux / macOS**
```bash
source face_blur/bin/activate
```

**Windows**
```bash
face_blur\Scripts\activate
```

## Usage

```bash
python face_blur.py
```

Press `q` to quit.
